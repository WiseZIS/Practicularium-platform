# PRACTICULARIUM Platform
# Database

## Назначение

Данный раздел содержит структуру базы данных проекта PRACTICULARIUM.

Все изменения базы данных оформляются в виде SQL-миграций и являются частью исходного кода проекта.

---

## Правила

1. Каждая миграция получает собственный порядковый номер.
2. Выполненные миграции никогда не изменяются.
3. Любые новые изменения оформляются новым SQL-файлом.
4. Миграции выполняются последовательно.
5. История изменений базы данных хранится вместе с проектом.
